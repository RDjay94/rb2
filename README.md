# RajaBaji 2.0 — Interactive Wireframe

**Live preview:** https://rdjay94.github.io/rb2/

Internal prototype for the 2026 RajaBaji Android app revamp. 22 landscape screens, gaming-app interactivity (spinning wheel, animated reels, flying plane, confetti, ticking counters), and real game tile art.

## Access
Password-protected (ask RD Jay).

## Stack
- Single-file HTML + Tailwind via CDN
- Vanilla JS for interactivity (no framework)
- Brand palette: `#9DE134` brand green · `#0A0A0A` jet black · neutrals 10/60/30 ratio
- Frame ratio: 19.5:9 landscape (884 × 408)
